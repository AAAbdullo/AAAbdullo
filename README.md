### 👨‍💻 Обо мне

Я — специалист отдела разработки программных продуктов в государственном научно-исследовательском институте. Занимаюсь созданием технологичных решений, уделяя внимание архитектуре и качеству интерфейсов.

- 📱 **Специализация:** Кроссплатформенная мобильная разработка на **Flutter** и **Dart**
- 🏗 **Архитектура:** Использую современные подходы (**BLoC**, **Riverpod**) для создания надёжных систем
- 🧪 **Исследования:** Работаю над проектами автоматизации и цифровизации в государственном секторе
- 🌱 **Сейчас изучаю:** **Python** для задач автоматизации и бэкенд-разработки
